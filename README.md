# Cliente
### Descripción:

El archivo client.c implementa un cliente de socket TCP simple que se conecta a un servidor especificado por la dirección IP y el puerto proporcionados como argumentos de línea de comandos. Después de establecer la conexión, el cliente envía un mensaje al servidor y espera una respuesta.

# Servidor
### Descripción
El archivo server.c es un ejemplo básico de un servidor de socket TCP. Espera conexiones en un puerto especificado como argumento de línea de comandos. Cuando un cliente se conecta, el servidor acepta la conexión, envía un mensaje de confirmación al cliente y luego cierra la conexión.
